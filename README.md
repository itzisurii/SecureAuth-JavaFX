# SecureAuth-JavaFX

## About
SecureAuth-JavaFX is a JavaFX desktop application that demonstrates a secure user authentication system. It supports user registration and login with strict `@gmail.com` validation and strong password rules. Passwords are hashed using **SHA-256** and are never stored in plain text.

---

## Features
- JavaFX-based desktop user interface  
- User registration and login system  
- Email validation restricted to `@gmail.com`  
- Strong password validation rules  
- Secure password hashing using SHA-256  
- Clean separation of UI and authentication logic  

---

## Technologies Used
- Java  
- JavaFX  
- SHA-256 for password hashing  
- Scene Builder   
- IntelliJ IDEA   

---

## Password Rules
Passwords must:
- Meet the minimum required length  
- Include a mix of uppercase and lowercase letters  
- Contain numbers and special characters  

> **Note:** Plain-text passwords are never stored or logged.
